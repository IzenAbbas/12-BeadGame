# 12-BeadGame
## Project Overview
The "12 Beads" is a two-player strategy board game where the goal is to eliminate all of the opponent's beads. The game uses a 25-slot board with 16 mini-boxes. Each player controls 12 beads and moves them according to specific rules. Players take turns to eliminate the opponent's beads by landing on vacant positions after crossing over an opponent's bead.

## Game Rules
#### Players: 
2 players, each controlling 12 beads of different colors (e.g., red and green).

#### Board Setup:
The board consists of 25 positions arranged in a diamond shape with an empty center position.

#### Movement: 
Players take turns moving their beads along vertical or horizontal lines.

#### Elimination: 
A player can eliminate an opponent's bead if their bead lands on an empty position just after the opponent's bead.

#### Winning: 
The first player to eliminate all of the opponent's beads wins the game.


## GUI: 
SFML is used for the implementation of GUI of this game.
